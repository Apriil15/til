# Toggle

研究這個是因為 notion export 的 Markdown file 不會有 toggle。目前看起來要自己處理，希望未來 notion 有支援 R

格式如下：

```markdown
<details>
  <summary>title</summary>
  
  - content

</details>
```

注意：

- summary 要記得縮排
